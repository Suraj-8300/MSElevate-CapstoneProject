## 📌 Project Overview
This project, developed as part of the **Microsoft Elevate AICTE Internship Capstone Project**, focuses on analyzing retail sales data from Big Mart. The primary goal is to transform large volumes of raw data into meaningful business insights to identify high-performing products, efficient outlet types, and location-based sales trends.

## 📊 Data Analysis (DAX)
Calculated key metrics to drive the dashboard:
* **Average Sales:** 2.18K
* **Number of Products:** 1,559
* **Number of Stores:** 10

## 🎨 Data Visualization
Created interactive charts including:
* **Sales by Item Type:** Bar Chart
* **Sales by Outlet Type:** Horizontal Bar Chart
* **Sales Distribution by Outlet Size:** Pie Chart
* **Sales by Location Type:** Column Chart

## 💻 System Requirements
To view or edit the dashboard, the following specifications are recommended:
* **OS:** Windows 10 or higher
* **RAM:** Minimum 8 GB
* **Storage:** 10 GB free space

## 🚀 Future Scope
* **Real-time Data Integration:** Connection to live sales databases.
* **Sales Forecasting:** Predictive analytics for demand estimation.
* **Customer Behavior Analysis:** Pattern analysis using Machine Learning.
* **Cloud Integration:** Scaling via Power BI Service.
